# Input Label

A styled version of <a href="https://mui.com/material-ui/react-typography/" target="_blank">MUI's Typography</a>, so you can use any prop provided there.

## Usage

- Always wrap with a `FormControl` component in order to use `useFormControlUnstyledContext` hook to determine if input is invalid.

## Example

```
import React, { useState } from 'react';
import { FormControl, InputError } from '@moderntribe/wme';

export const MyInputError = (props) => (
  <FormControl>
    <InputError>
      This field is required.
    </InputError>
  </FormControl>
);

export default InputError;
```
