# Input Label

A styled version of <a href="https://mui.com/material-ui/api/form-helper-text/" target="_blank">MUI's FormHelperText</a>, so you can use any prop provided there.

## Example

```
import React, { useState } from 'react';
import { InputHelperText } from '@moderntribe/wme';

export const MyInputHelperText = (props) => (
  <InputHelperText>The Doors of Durin, also known as the West-gate, the West-door of Moria, or Elven Door, were built into the Walls of Moria in the dark cliffs.<InputHelperText>
);

export default MyInputHelperText;
```
