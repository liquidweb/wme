# Input Label

A styled version of <a href="https://mui.com/base/react-form-control/" target="_blank">MUI's FormControlUnstyled</a>, so you can use any prop provided there.

## Example

```
import React, { useState } from 'react';
import { FormControl } from '@moderntribe/wme';

export const MyFormControl = (props) => (
  <FormControl>
    ...
  <FormControl>
);

export default MyFormControl;
```
