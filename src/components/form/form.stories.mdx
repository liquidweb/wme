# Form

A styled version of <a href="https://mui.com/material-ui/react-box" target="_blank">MUI's Box</a>, so you can use any prop provided there.

## Props

<ArgsTable story="." />

## Example

```
import React from 'react';
import { Form, FormField, TextInput } from '@moderntribe/wme';

export const MyForm = (props) => (
  <Form>
    <FormField
      field={<TextInput />}
      label="Text Input"
    />
  </Form>
)

export default Form;
```
