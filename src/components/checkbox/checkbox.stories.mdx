# Checkbox

Composes `FormControlLabel` and `CheckboxInput` to create a single, re-usable "checkbox" component.

**_Note:_** Any additional props passed to this component will be passed to <a href="https://mui.com/material-ui/api/form-control-label" target="_blank">MUI's FormControlLabel</a>, so you can use any prop provided there.


## Usage

- Checkboxes should be used independently from each other.
- Ensure both label and input are clickable to select the checkbox field.
