# Input Label

A styled version of <a href="https://mui.com/material-ui/api/input-label/" target="_blank">MUI's InputLabel</a>, so you can use any prop provided there.

## Example

```
import React, { useState } from 'react';
import { InputLabel } from '@moderntribe/wme';

export const MyInputLabel = (props) => (
  <InputLabel>Input Label<InputLabel>
);

export default MyInputLabel;
```
