# Chip

Chips are used to display quick information to the user and should always be
paired with a color that helps clearly communicate the information.

## Example

```
import React from 'react';
import { Chip } from '@moderntribe/wme';

const myChip = () => {
  <>
    <h1>Chip Component</h1>
    <Chip
      color="success"
      size="small"
      icon={<AccessTime />}
    />
  </>
};

export default myChip;
```
