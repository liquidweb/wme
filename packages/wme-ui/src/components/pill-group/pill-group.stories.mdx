# Pill Group

A PillGroup will handle a group of Pills, formatting them and returning a list of updated active statuses on every select.

## Props

<ArgsTable story="." />

## Example

```
import React from 'react';
import { PillGroup } from '@moderntribe/wme';

const myPillGroup = () => {
  <>
    <h1>Pill Group</h1>
    <PillGroup
		pills={[
			{
				label: 'Quirky',
				active: false
			},
			{
				label: 'Calm',
				active: false
			}
		]}
		onSelect={(pills) => console.log(pills)}
    />
  </>
};

export default myPillGroup;
```
