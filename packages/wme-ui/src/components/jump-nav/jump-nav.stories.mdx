# Jump Navigation
Component displaying a list of anchor links for creating a navigation in a single page for the sections.

Links should be passed in as an array of objects, as modeled below.

```
const links = [
	{
		id: 0,
		label: "label 1",
		href: "#href-1",
	},
    {
        id: 1,
        label: "label 2",
        href: "#href-2",
        remainingTasks: 3
    },
];
```
